# zira-game

## Instalation
*  Clone this repository:
```sh
git clone https://github.com/jamasellipsis/zira-game.git
```
* Inside the repository execute:
```
npm install
```
* Open two terminals:
* For the first terminal execute:
```sh
node main.js
```
* For the other terminal execute:
```
npm i -g peer
```
```
peerjs --port 3001
```
